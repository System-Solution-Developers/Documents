プライバシーポリシー

最終更新日：2026年4月27日　バージョン：1.0

ほんとも（以下「本アプリ」）の提供者（以下「運営者」）は、ユーザーのプライバシー保護を最重要事項と位置づけ、個人情報の保護に関する法律（個人情報保護法）を遵守し、以下のプライバシーポリシー（以下「本ポリシー」）に従い、適切な取り扱いおよび保護に努めます。

第1条（個人情報の定義）

本ポリシーにおいて「個人情報」とは、個人情報保護法に定める個人情報を指し、生存する個人に関する情報であって、氏名、生年月日、住所、電話番号、メールアドレスその他の記述等により特定の個人を識別できる情報を意味します。

第2条（収集する情報と保存場所）

本アプリは、ユーザーの個人情報を一切収集しません。アカウント登録・ログインは不要であり、氏名・メールアドレス・電話番号等の個人を特定できる情報を取得することはありません。

本アプリに記録する読書データ（本の情報、読書セッション、学び、ハイライト等）はすべてユーザーのデバイス内にのみ保存されます。外部サーバーへの送信・同期は一切行いません。

第3条（カメラの利用）

  1. 本アプリはISBNバーコードのスキャン機能のためにカメラへのアクセス許可を求める場合があります。
  2. カメラは書籍のISBNバーコード読み取りにのみ使用します。撮影された画像は外部サーバーへ送信・保存されることはなく、バーコードの数値情報のみが書籍検索に使用されます。
  3. カメラ機能の利用はユーザーの任意です。許可しない場合でも、タイトルや著者名のテキスト入力による書籍検索が利用できます。
  4. 許可設定はいつでもiOSのシステム設定から変更できます。

第4条（通知機能の利用）

  1. 本アプリは任意で設定できる読書リマインダー通知のために、通知の送信許可を求める場合があります。
  2. 通知はすべてローカル通知であり、外部サーバーへの情報送信を伴いません。
  3. 通知機能の利用はユーザーの任意です。通知を無効にしても本アプリの主要機能に影響はありません。
  4. 許可設定はいつでもiOSのシステム設定またはアプリ内設定から変更・解除できます。

第5条（外部サービスへの情報送信）

本アプリは書籍情報の検索のため、以下の外部サービスのAPIを利用しています。

サービス名 送信される情報 利用目的 プライバシーポリシー
Google Books API
（Google LLC）		 書籍タイトル・著者名・ISBNコード（検索クエリ） 書籍情報・書影の取得 Google プライバシーポリシー
OpenBD API

上記APIへの送信内容は書籍の検索クエリのみです。ユーザーの氏名・メールアドレス等の個人を特定できる情報は一切送信されません。また、各APIサービスにおける情報の取り扱いについては、各サービスのプライバシーポリシーをご確認ください。

第6条（広告・分析ツールの不使用）

本アプリのMVP（初期）バージョンでは、以下のサービスを一切使用していません。

  • 広告配信SDK（AdMob等）
  • 行動分析ツール（Firebase Analytics、Mixpanel、PostHog等）
  • クラッシュ解析ツール（Sentry等）
  • 外部認証サービス（Apple ID、Google アカウント等）

将来的に上記サービスを導入する場合には、事前に本ポリシーを改定のうえ、ユーザーへ周知いたします。

第7条（データのエクスポート）

  1. 本アプリはユーザー自身の読書データをJSON・CSV・Markdown形式でエクスポートする機能を提供しています。
  2. エクスポートはすべてユーザーの任意の操作によるものであり、運営者がユーザーデータを自動的に収集・送信することはありません。
  3. エクスポートされたファイルの管理・共有はユーザー自身の責任において行われます。

第8条（個人情報の安全管理）

本アプリはユーザーのすべての読書記録データをデバイス内のSQLiteデータベースに保存します。データは端末の外部へ送信されないため、外部からの情報漏洩リスクを最小化した設計となっています。デバイス自体のセキュリティ（画面ロック等）を適切に設定することをお勧めします。

第9条（データの削除）

  1. ユーザーはアプリ内の操作から、個別の本・読書セッション・学び・ハイライトをいつでも削除できます。
  2. 本アプリをアンインストールすることで、デバイス内に保存されたすべての読書記録データが削除されます。
  3. 削除されたデータの復元は行えません。機種変更等の際は、エクスポート機能をご利用ください。

第10条（お子様のご利用）

本アプリは個人情報を収集しないため、年齢に関わらずご利用いただけます。ただし、13歳未満のお子様がご利用される場合は、保護者の方の監督のもとでのご利用を推奨いたします。

第11条（プライバシーポリシーの変更と改訂履歴）

  1. 本ポリシーの内容は、法令その他本ポリシーに別段の定めのある事項を除いて、変更することができるものとします。
  2. 変更後のプライバシーポリシーは、本アプリまたは本ページに掲載した時点から効力を生じるものとします。
  3. 重要な変更がある場合には、アプリ内通知等の適切な方法によりユーザーへ周知いたします。
